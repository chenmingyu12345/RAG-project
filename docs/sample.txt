DeepSeek API 使用指南

DeepSeek 是一个强大的人工智能模型服务平台，提供文字生成、代码辅助等功能。

接入方式
1. 注册账号并获取 API Key
2. 使用 OpenAI 兼容接口调用
3. 支持 Chat Completions 和 Embeddings 接口

Chat Completions
DeepSeek 的对话补全接口与 OpenAI 完全兼容。可以使用 langchain-openai 库来调用，只需将 base_url 指向 https://api.deepseek.com 即可。

Embeddings
DeepSeek 提供文本向量化接口，可用于语义搜索、文档聚类等场景。

定价说明
DeepSeek 提供极具竞争力的价格，相比同类产品具有显著优势。具体价格请参考官方网站。

注意事项
- 注意 API 调用频率限制
- 妥善保管 API Key
- 选择合适的模型版本
